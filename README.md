# THBank-Scraping

## Description

Scraping total balance in your bank account from internet banking.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```
